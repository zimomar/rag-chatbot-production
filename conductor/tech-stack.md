# Tech Stack - UI Refinement

## Frontend Framework
- **Streamlit**: Core framework for speed of deployment and integration with Python RAG backends.
- **Custom CSS (SCSS/CSS)**: Deep customization of Streamlit primitives.
- **Lucide/Material Icons**: For a consistent and modern iconography set.
- **Inter/Geist/Monospace Fonts**: Premium font pairings.

## Enhancements
- **Lottie/CSS Animations**: For micro-interactions (e.g., subtle loading indicators, hover states).
- **Glassmorphism**: Advanced CSS filters (`backdrop-filter`).
- **Dynamic Themes**: Enhanced dark/light logic with custom color palettes (e.g., Slate & Indigo).

## Visualization
- **Mermaid.js**: If we need dynamic diagramming (already supported in Streamlit, but could be better styled).
- **Custom HTML/JS Components**: For any features Streamlit can't handle with standard widgets.
